# Robot Command Simplifier

This program simplifies command sequences for a robot, generating a new command sequence that guides the robot with the fewest movements possible. The program consists of functions to simplify individual commands and entire paths.

## Instructions

1. **simplifyCommand**: This function takes a command sequence as input and returns a simplified version of the sequence by consolidating repeated characters.

2. **simplifyPath**: This function takes a path as input and returns a simplified version of the path by combining movements along the x and y axes.

## Usage
1. Compile the program using a C++ compiler.
2. Run the compiled executable.
3. Input the robot's commands when prompted.
4. The program will output the simplified command sequence.
